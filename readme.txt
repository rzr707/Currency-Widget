Версия приложения 05.12.2016 (ver. 0.2.2)
1) Добавил сигнал к классу CurrencyWidget, который посылается тогда,
   когда успешно записываются новые данные в файлы чартов
2) Чарты теперь записываются не в корневую папку с .ехе, а в папку
   /charts/, где создаётся подпапка для каждого дня (если не создана),
   и уже в ней создаются файлы чартов
3) В Tray добавил возможность загрузить окно со старым чартом
4) Теперь, при первом запуске программы будет запускаться
   сообщение с краткой инструкцией по пользованию программой
   (такая же, как в Tray->Инструкция)
5) Сделал проверку в void slotFinished(QNetworkReply* reply).
   Если ответ на запрос пришёл с ошибкой (или вообще не пришёл),
   то программа аварийно завершается, предупредив об этом пользователя
6) Проведён небольшой рефакторинг current.h
7) Добавлена возможность выбирать отображение купли или продажи валюты ПриватБанком
8) Добавлена возможность выбирать отображение купли/продажи наличной и безналичной валюты.
9) Для п.7 и 8 был переписан метод парсинга запроса апи, а также несколько других
   методов, связанных с этим
10) Добавлены 2 переменные, для хранения статуса типа валюты(нал/безнал) и купли/продажи
11) Добавлена сериализация/десериализация переменных из пункта 10
